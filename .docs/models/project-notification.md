# プロジェクト通知

プロジェクト通知はプロジェクト内で発生した重要なイベントに関する情報を表すエンティティです。プロジェクト内の活動や変更について、関係者に適切なタイミングで知らせるために使用されます。

## 種類

エンティティ

## 属性

**id**
- 型: string
- 説明: 通知を一意に識別するID
- 制約: UUID形式

**projectId**
- 型: string
- 説明: 通知が関連するプロジェクトの識別子
- 制約: 有効なプロジェクトIDである必要がある

**title**
- 型: string
- 説明: 通知の見出し
- 制約: 空でない文字列

**message**
- 型: string
- 説明: 通知の詳細内容
- 制約: 空でない文字列

**type**
- 型: string
- 説明: 通知の種類
- 制約: "INFO", "WARNING", "ERROR", "SUCCESS"のいずれか

**createdAt**
- 型: Date
- 説明: 通知が作成された日時

**deletedAt**
- 型: Date | null
- 説明: 通知が削除された日時（論理削除）
- 制約: nullの場合は削除されていない

## ビジネスルール

- 通知のタイプは"INFO", "WARNING", "ERROR", "SUCCESS"のいずれかでなければならない
- 通知の削除は論理削除（deletedAtに日時を設定）で行う
- タイトルとメッセージは空であってはならない

## 操作

- 通知の削除: deletedAtに現在時刻を設定する

## 関連

- プロジェクト通知は1つのプロジェクトに所属する（多対1）

## 通知例

- メンバー追加: 新しいメンバーがプロジェクトに追加された
- 設定変更: プロジェクト設定が変更された
- 期限通知: プロジェクトの期限が近づいている
- エラー通知: 処理中にエラーが発生した
